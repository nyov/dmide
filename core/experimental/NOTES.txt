Newly created widgets for DMIDE are found here.
Extensive testing appreciated!
Also looking for screenshots of these in action.


REQUIRED:

You'll need to install Python first, version 2.3 to 2.6 should work.
http://www.python.org/download/

PIL comes second. version 1.1.5 and above should work.
http://www.pythonware.com/products/pil/

PIL PNG zTXt Patch required. 
http://mail.python.org/pipermail/image-sig/2007-February/004343.html
Patched file included. Replace `[Python installed dir]/Lib/site-packages/PIL/PngImagePlugin.py` with included file.

wxPython comes third. version 2.8.9.1 recommended.
http://wxpython.org/download.php
